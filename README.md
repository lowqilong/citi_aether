# citi_aether

## Backend APIs

### Prerequisites

1. Copy paste .env.example in the same folder. Rename to .env.

2. Add your AWS credentials into the .env file.

3. Go to backend folder (cd backend)

4. Install packages (npm i)

5. Run the application.

For hot-reloading

```
npm run dev
```

No hot-reloading

```
npm run start
```
